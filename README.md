# 🌌 Mirza Shaheer - Cinematic Portfolio

Welcome to my personal cinematic portfolio repository! This is a highly polished, single-page web application designed to showcase my projects, skills, and engineering journey. It features a dark, immersive aesthetic with fluid animations, dynamic typography, and a premium modern design language.

## ✨ Key Features

- **Single-File Architecture:** The entire portfolio logic, structure, and styling are elegantly contained within a single `index.html` file. All graphical assets, including project screenshots and icons, are encoded as Base64 strings to ensure an entirely self-contained deployment.
- **Cinematic Aesthetic:** Incorporates advanced CSS techniques like subtle film grain overlays, custom vignette effects, glowing orbs, and mouse-following spotlights to create an ambient, immersive user experience.
- **Responsive Design:** A fully responsive layout that adapts seamlessly across desktop, tablet, and mobile devices with precise typography scaling.
- **Dynamic Navigation:** Features a robust, custom-built scroll-spy navigation system in Vanilla JavaScript that automatically highlights the active section as the user scrolls.

## 🛠️ Technology Stack

- **HTML5:** Semantic, accessible page structure.
- **CSS3:** Advanced styling, custom animations (`@keyframes`), CSS variables, glassmorphism, and CSS Grid/Flexbox layouts. Built entirely with Vanilla CSS for maximum control and performance without framework overhead.
- **Vanilla JavaScript:** Powers the active scroll-tracking logic and interactive DOM elements.

## 🚀 Portfolio Sections

- **Hero:** A high-impact introduction combining dynamic gradient typography and an elegant sub-headline.
- **About:** A professional summary featuring a downloadable interactive CV integrated into an ambient rotating orb.
- **Work:** Detailed project cases highlighting technical achievements in my top projects including *NUsphere*, *NuGames*, *NuQuizzer*, and a *Custom C Shell*.
- **Education:** Academic background and institutional achievements.
- **Contact:** Quick access to professional networks (LinkedIn, GitHub) and a direct email call-to-action.

## 🔗 Live Preview

## ⚙️ Setup & Deployment

Due to its self-contained architecture, interacting with and deploying this project is exceptionally straightforward:

1. Clone or download this repository.
2. Open `index.html` in any modern web browser to view the site locally.
3. Deploy instantly to any static hosting service (like **Vercel**, **Netlify**, or **GitHub Pages**) by simply connecting this repository.
